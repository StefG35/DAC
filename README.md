# Deltarune ALL CHOICES
Un Canvas di Obsidian per racchiudere tutte le scelte a medio/lungo termine di Deltarune

## REGOLE PRINCIPALI
1. Le scelte facoltative creano un ramo DI LATO alla cella, invece le scelte obbligate si creano verso il basso
2. I colori servono per le scelte a medio/lungo termine in modo da poter avere collegamenti che si intersecano senza confordersi
3. Il verde predefinito di Obsidian si usa per le scelte multiple come il *nome del giocatore* e il *PestAggeggio*
4. Si dà per scontato il dialogo che appare senza la scelta facoltativa
5. I collegamenti colorati (tranne quello per la Weird Route) non "proseguono" la trama principale, sono delle condizioni da rispettare per ottenere quel risultato 
6. Mantieni una coerenza linguistica, cerca di utilizzare sempre l'italiano e il "glossario" della traduzione di Undertale Spaghetti Project
7. Usa screenshot provenienti dal gioco moddato in italiano
